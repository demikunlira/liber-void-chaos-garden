# Character Reference Bible Creator

**Purpose:** Create clean, consistent, professional Character Reference Bibles that lock visual canon for every character and Aspect in the Liber Void Garden.

**When to use:** Whenever a new character or Aspect needs official references, or when updating existing ones for consistency.

## What It Creates

- A dedicated folder inside `Character reference bibles/`
- A master PDF with multiple pages:
  - Full body front/side/back
  - Close-up face (neutral, smiling, serious, power-activated)
  - Hand close-ups (especially sigil/glow details)
  - Expression sheet (6–8 key emotions)
  - Outfit variations (casual, combat, formal, Garden form)
- 8–12 high-quality reference images in consistent style

## Workflow

1. Provide character name + detailed description + any existing reference images
2. The skill generates a complete, consistent set
3. Everything is saved in the correct folder with proper naming
4. A summary PDF is created for easy sharing

## Important Rules

- All images must match the God in Pink aesthetic (90s cel-shaded, pink neon accents, glowing effects where appropriate)
- Maintain strict consistency — no drifting between images
- Include the character's signature glow/mark (Isaac’s palm sigil, Demikun’s pink streaks, etc.)
- Make references usable for both manga production and future AI generation prompts
